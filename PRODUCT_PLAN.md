# Monn 股票分析软件 — 产品规划文档

> 版本：v1.0  
> 定位：个人投资者的一站式智能股票分析工具  
> 市场：港股 / 美股  
> 核心：行情数据 + 技术分析 + 基本面 + AI 智能决策辅助  
> 不含实盘交易，仅做行情数据分析、预测研判与模拟持仓

---

## 产品定位

Monn 是一款面向个人投资者的**专业级复盘、选股、技术基本面分析工具**，聚焦港股与美股市场。围绕"看盘→分析→选股→决策→复盘"全链路，提供实时行情、技术指标、财务数据、资金筹码、AI 智能研判与模拟持仓等功能。

**核心差异化：**
- AI 驱动的个股走势研判与买卖建议
- 模拟持股 + AI 策略回测的闭环验证
- 直觉化现代 UI，降低专业分析工具的学习门槛

---

## UI 设计理念

- **暗色主题为主**，护眼且适合长时间盯盘，参考 TradingView / Bloomberg Terminal 的现代扁平风格
- **模块化仪表盘**：首页支持拖拽自定义布局，每个用户可配置自己的工作台
- **数据可视化优先**：K 线图、热力图、雷达图、资金流向桑基图等交互式图表
- **响应式适配**：桌面端为主，平板为辅，移动端提供精简看盘版
- **多窗口/多标签**：支持同时打开多只股票、多个时间周期对比
- **键盘快捷键**：专业用户可纯键盘完成切换股票、切换周期、调出指标等高频操作

## 九、AI 智能分析模块

### 9.1 AI 走势研判
| 功能点 | 说明 |
|--------|------|
| 短期走势预测 | 基于多维度数据（技术+资金+情绪），AI 给出 1-5 日走势倾向判断 |
| 中期趋势研判 | 基于基本面+行业+宏观，AI 给出 1-3 个月趋势方向 |
| 置信度标注 | 每次研判附带置信度百分比，不夸大预测能力 |
| 多空信号 | 综合多指标信号的多空评分（0-100分） |
| 拐点预警 | AI 识别潜在的趋势拐点（顶部/底部信号） |

### 9.2 风险识别
| 功能点 | 说明 |
|--------|------|
| 财务风险 | 识别高负债、现金流恶化、应收账款异常、商誉减值风险 |
| 市场风险 | 估值过高、行业政策利空、流动性风险 |
| 技术面风险 | 破位风险、顶背离、放量滞涨等 |
| 事件风险 | 大股东减持、诉讼、监管问询、业绩变脸 |
| 综合风险评分 | 0-100 风险评分，多维度雷达图展示风险来源 |

### 9.3 涨跌原因总结
| 功能点 | 说明 |
|--------|------|
| 日度涨跌归因 | 综合新闻、公告、资金、板块联动，AI 总结当日涨跌主要原因 |
| 归因权重 | 每个影响因素分配权重百分比（如"财报超预期 40%，板块带动 30%...") |
| 异常波动解析 | 对盘中异常放量/急涨急跌给出可能原因分析 |

### 9.4 个股精简分析报告
| 功能点 | 说明 |
|--------|------|
| 一键生成报告 | 输入股票代码，AI 自动生成一份精简分析报告 |
| 报告内容 | 公司简介 → 技术面概览 → 基本面亮点/风险 → 资金面 → 综合评分 → 操作建议 |
| 多时间维度 | 短线视角 / 中线视角 双维度评估 |
| 关键价位 | AI 识别关键支撑位、压力位、目标价区间 |
| 报告导出 | PDF 导出 + 分享链接 |
| 历史报告 | 个股历史报告存档，可对比 AI 判断准确率 |

### 9.5 AI 股票推荐
| 功能点 | 说明 |
|--------|------|
| 智能推荐 | 基于用户自选股偏好+当前市场环境，AI 每日推荐 3-5 只关注标的 |
| 推荐理由 | 每只推荐附带详细逻辑说明（技术面+基本面+催化剂） |
| 推荐追踪 | 追踪历史推荐股票的表现，建立 AI 推荐可信度档案 |
| 风格适配 | 用户可选择偏好风格（短线激进/中线稳健/长线价值），AI 调整推荐策略 |
| 风险提示 | 每只推荐附带风险警示和止损建议 |

### 9.6 AI 策略回测
| 功能点 | 说明 |
|--------|------|
| 策略描述 | 用自然语言描述交易策略，AI 转化为回测规则 |
| 回测结果 | 收益率曲线、最大回撤、夏普比率、胜率、盈亏比 |
| 参数优化 | AI 自动遍历关键参数组合，寻找最优参数 |
| 过拟合警告 | AI 检测策略是否过度拟合历史数据 |
| 与模拟持仓联动 | 回测验证的策略可直接用于模拟账户自动执行（模拟） |

## 技术架构建议

| 层级 | 技术选型建议 |
|------|-------------|
| 前端 | Electron（桌面端）+ React + TypeScript + ECharts / TradingView Charting Library |
| 后端 | Go / Python (FastAPI)，WebSocket 推送实时行情 |
| 数据源 | 港股/美股 行情数据 API（如 Bloomberg、Wind、东方财富、Yahoo Finance） |
| AI 层 | LLM API（Claude / GPT）+ 本地轻量模型做指标计算 |
| 存储 | PostgreSQL（业务数据）+ InfluxDB（时序行情数据）+ Redis（缓存/实时推送） |
| 部署 | 桌面端本地运行 + 云端同步用户配置和自选股数据 |
